# Git Large File Storage (LFS)

## CLI

### Installation

#### Homebrew

```sh
brew install git-lfs
```

#### YUM

```sh
yum check-update

# Repo: EPEL
sudo yum -y install git-lfs
```

#### APT

```sh
sudo apt update
sudo apt -y install git-lfs
```

#### APK

```sh
sudo apk update
sudo apk add git-lfs
```

### Commands

```sh
man git-lfs
```

### Extend

#### With Trunk

```sh
trunk actions enable git-lfs
```
