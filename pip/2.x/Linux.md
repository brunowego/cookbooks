# Linux

## Installation

### YUM

```sh
sudo yum check-update
sudo yum -y install epel-release
sudo yum -y install python2-pip
```

> Preferably to install through [SCL](https://www.softwarecollections.org)

### APT

```sh
sudo apt update
sudo apt -y install python-pip
```

### APK

```sh
sudo apk update
sudo apk add py-pip
```
