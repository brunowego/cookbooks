## CLI

### Installation

#### Homebrew

```sh
brew install terraform
```

### Commands

```sh
terraform -h
```

### Tips

#### Visual Studio Code

```sh
# Install
code --install-extension mauve.terraform

## Disable
# code --disable-extension mauve.terraform
```
