## CLI

### Installation

#### Homebrew

```sh
brew install libsass sassc
```

#### APT

```sh
sudo apt update
sudo apt -y install libsass-dev pysassc
```

#### YUM

```sh
yum check-update

# Repo: EPEL
sudo yum -y install libsass-devel sassc
```
