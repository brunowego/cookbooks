## CLI

### Installation

```sh
# Using NPM
npm install dotenv-cli -g

# Using Yarn 1.x
yarn global add dotenv-cli
```

### Commands

```sh
# Using local
dotenv -h
```

### Usage

```sh
#
dotenv -- <command>
```
