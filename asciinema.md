# asciinema

## Links

- [Main Website](https://asciinema.org)

## CLI

### Installation

#### Homebrew

```sh
brew install asciinema
```

#### YUM

```sh
yum check-update
sudo yum -y install asciinema
```

#### APT

```sh
sudo apt update
sudo apt -y install asciinema
```

### Commands

```sh
asciinema -h
```

### Configuration

```sh
#
asciinema auth
```

### Usage

```sh
#
asciinema rec

#
asciinema play ./<filename>.cast
```
