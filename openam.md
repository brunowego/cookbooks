## Docker

### Network

```sh
docker network create workbench \
  --subnet 10.1.1.0/24
```

### Running

```sh
docker run -d \
  $(echo "$DOCKER_RUN_OPTS") \
  -h openam \
  -v openam-config:/usr/openam/config \
  -p 8080:8080 \
  --name openam \
  --network workbench \
  docker.io/openidentityplatform/openam:14.5.3
```

> Wait! This process take a while.
