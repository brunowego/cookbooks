# CMake

## Links

- [Main Website](https://cmake.org/)

## CLI

### Installation

#### Homebrew

```sh
brew install cmake
```

#### APT

```sh
sudo apt update
sudo apt -y install cmake
```

#### YUM

```sh
yum check-update
sudo yum -y install cmake
```

#### APK

```sh
apk add cmake
```

### Commands

```sh
cmake -h
```
