# iputils

## Installation

### APT

```sh
sudo apt update

# ping
sudo apt -y install iputils-ping

# arping
sudo apt -y install iputils-arping

# clockdiff
sudo apt -y install iputils-clockdiff

# tracepath
sudo apt -y install iputils-tracepath
```

### YUM

```sh
yum check-update
sudo yum -y install iputils
```

### APK

```sh
sudo apk update
sudo apk add iputils
```
