## Tips

### PNG to ICO

**Dependencies:** [ImageMagick](/imagemagick.md)

```sh
#
convert \
  -resize x32 \
  -gravity center \
  -crop 32x32+0+0 \
  ./favicon.png \
  -flatten \
  -colors 256 \
  -background transparent \
  ./favicon.ico

#
convert ./favicon.png \
  -background white \
  \( -clone 0 -resize 32x32 -extent 32x32 \) \
  -delete 0 \
  -alpha off \
  -colors 16 \
  ./favicon.ico
```
