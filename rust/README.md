## CLI

### Installation

#### Homebrew

```sh
brew install rust
```

#### APT

```sh
sudo apt update
sudo apt -y install rustc
```

#### YUM

```sh
yum check-update

# Repo: EPEL
sudo yum -y install rust
```

### Commands

```sh
rustc --help
```

### Usage

```sh
# Version
rustc --version | head -1
```

### Tips

#### Visual Studio Code
