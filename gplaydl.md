## CLI

### Dependencies

On Python3 version 3.8.14.

### Installation

#### Homebrew

```sh
pip3 install gplaydl
```

### Commands

```sh
gplaydl -h
```

### Configuration

```sh
gplaydl configure
```

### Usage

```sh
#
gplaydl download \
  --packageId com.twitter.android \
  --path ./apk-downloads/
```
