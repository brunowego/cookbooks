# What

**Keywords:** Identify Anything, Hash Identifier

## Links

- [Code Repository](https://github.com/bee-san/pyWhat)

## CLI

### Installation

#### Homebrew

```sh
brew install pywhat
```

#### pip

```sh
pip3 install pywhat
```

### Commands

```sh
pywhat --help
```

### Usage

```sh
# Use with quotation
pywhat '0x52908400098527886E0F7030069857D2E4169EE7'
# or, POSIX standard
pywhat -- 0x52908400098527886E0F7030069857D2E4169EE7
```
