### Issues

#### Missing Git CLI

```log
'git status' failed with code -1:'launch path not accessible
```

1. Preferences
2. Git tab
3. Git Version -> Use System Git

#### Authentication Required

```log
Authentication Required
For URL: https://[]/[].git
```

1. Sourcetree
2. Preferences...
   - Git tab
   - Git Version block -> Use System Git
