### Installation

```sh
# Using NPM
npm install @changesets/cli --save-dev

# Using Yarn
yarn add @changesets/cli --dev

# Using pnpm
pnpm add @changesets/cli -D
```

### Initialize

```sh
# Using Yarn
yarn changeset init

# Using pnpm
pnpm changeset init
```
