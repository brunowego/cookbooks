## Tips

### Create Application

1. [Dashboard](https://manage.auth0.com/dashboard) -> Applications -> Create Application
2. Name: `<App Name>` -> Regular Web Application -> Create
3. Settings -> Allowed Callback URLs: `http://localhost:3000/callback`
