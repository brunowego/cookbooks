# Correlation

## With US Dollar Index (DXY)

**Level:** Strong

1. Open `TVC:DXY`
2. Compare or Add Symbol `INDEX:BTCUSD` -> New price scale
3. Indicators `BTCUSD, INDEX` -> More -> Pin to scale -> Pinned to scale B
4. Toggle Log Scale
5. Indicators `BTCUSD, INDEX` -> Settings -> Style Baseline
