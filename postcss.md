## Tips

### Valid File Extensions

- `.css`
- `.pcss`
- `.postcss`

### Visual Studio Code

```sh
#
code --install-extension csstools.postcss

#
jq '."recommendations" += ["csstools.postcss"]' "$PWD/.vscode/extensions.json" | sponge "$PWD/.vscode/extensions.json"
```
