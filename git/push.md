# Git Push

## Usage

```sh
# Set Upstream
git branch --set-upstream-to=origin/<branch> <branch>

#
git push -u origin <local-branch>:<remote-branch>

#
git push origin <commit-sha>:<remote-branch> -f
```

## Tips

### Push to Remote using HEAD

```sh
git push origin HEAD
```
