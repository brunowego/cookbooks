## CLI

### Commands

```sh
npx preact-cli -h
```

### Global Installation

```sh
# Using NPM
npm install preact-cli -g

# Using Yarn 1.x
yarn global add preact-cli
```

### Usage

```sh
#
npx preact-cli create typescript [app-name]
```
