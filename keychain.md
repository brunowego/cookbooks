# Keychain

## App

### Commands

```sh
security help
```

### Usage

```sh
# Dump
sudo security dump-keychain /System/Library/Keychains/SystemRootCertificates.keychain

# Unlock Keychain
security \
  -v unlock-keychain \
  -p '<password>' \
  ~/Library/Keychains/login.keychain-db
```

### Tips

#### Trust Darwin

[Export Web Self-signed Certificate](/openssl.md#export-web-self-signed-certificate).

```sh
# Test
curl -v https://<hostname>

curl \
  --cacert ./selfsigned_certificate.pem \
  -v \
  https://<hostname>

# Or
wget \
  -O - \
  https://<hostname>

wget \
  -O - \
  --ca-certificate=./selfsigned_certificate.pem \
  https://<hostname>

# Add
sudo security add-trusted-cert -d \
  -r trustRoot \
  -k /Library/Keychains/System.keychain \
  ./selfsigned_certificate.pem

# Test
curl https://<hostname>

# NOTE: trustAsRoot

# Delete
openssl x509 \
  -in ./selfsigned_certificate.pem \
  -noout \
  -subject

sudo security delete-certificate -c '<hostname>'
```
