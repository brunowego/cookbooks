### Visual Studio Code

```sh
#
code --install-extension thibault-vanderseypen.i18n-json-editor

#
jq '."recommendations" += ["thibault-vanderseypen.i18n-json-editor"]' "$PWD"/.vscode/extensions.json | sponge "$PWD"/.vscode/extensions.json
```
