# asdf kOps

## Links

- [Code Repository](https://github.com/Antiarchitect/asdf-kops)

## Installation

```sh
asdf plugin-add kops 'https://github.com/Antiarchitect/asdf-kops.git'
```

## Usage

```sh
#
asdf list-all kops

#
asdf install kops [version]

#
asdf global kops [version]

#
asdf list kops

#
asdf reshim kops [version]
```
