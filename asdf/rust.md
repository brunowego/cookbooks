# asdf Rust

<!--
https://github.com/sdepold/rsvm
-->

## Installation

```sh
asdf plugin-add rust 'https://github.com/code-lever/asdf-rust.git'
```

## Usage

```sh
#
asdf list-all rust

#
KEEP_SOURCE=1 \
  asdf install rust <version>

#
asdf global rust <version>

#
asdf list rust

#
asdf reshim rust <version>
```

## Test

```sh
#
rustc --version

#
cargo --version
```
