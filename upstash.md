## CLI

### Installation

```sh
# Using NPM
npm install @upstash/cli -g
```

### Commands

```sh
upstash -h
upstash redis -h
```

### Configuration

```sh
upstash auth login --email <email> --api-key <api-key>
```

### Usage

```sh
#
upstash redis list
```
