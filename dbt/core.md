### Installation

#### Homebrew

```sh
brew tap dbt-labs/dbt
brew install dbt
# Or, for postgres
brew install dbt-postgres
```

### Commands

```sh
dbt -h
```

### Usage

```sh
#
dbt init <name>

#
dbt run
```
