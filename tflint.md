# TFLint

**Keywords:** Terraform Linter

## Links

- [Main Website](https://github.com/terraform-linters/tflint)

## CLI

### Installation

#### Homebrew

```sh
brew install tflint
```

#### Chocolatey

```sh
choco install tflint
```

### Commands

```sh
tflint -h
```

### Usage

```sh
#
TFLINT_LOG=debug tflint
```
