## Library

### Initialization

```sh
npx @inlang/cli config init
```

## CLI

### Installation

```sh
# Using npm
npm install @inlang/cli -g
```

### Initialization

```sh
#
inlang config init
```

### Commands

```sh
# Using npx
npx @inlang/cli -h

# Using local
inlang -h
```
