## CLI

### Installation

#### Homebrew

```sh
brew tap go-task/tap
brew install go-task
```

#### Linux Binary

```sh
curl -L 'https://github.com/go-task/task/releases/download/v3.4.3/task_linux_amd64.tar.gz' | \
  tar -xzC /usr/local/bin task
```

#### Chocolatey

```sh
choco install -y go-task
```

### Commands

```sh
task -h
```
