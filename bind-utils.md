# BIND Utilities

## CLI

### Installation

#### YUN

```sh
yum check-update
sudo yum -y install bind-utils
```

#### APT

```sh
sudo apt update
sudo apt -y install bind9utils
```

#### APK

```sh
sudo apk update
sudo apk add bind-tools
```

#### Chocolatey

```sh
choco install -y bind-toolsonly
```

dig -h
