# BZip2

## Links

- [Main Website](https://sourceware.org/bzip2/)

## CLI

### Installation

#### Homebrew

```sh
brew install bzip2
```

#### YUM

```sh
yum check-update
sudo yum -y install bzip2-devel
```

#### APT

```sh
sudo apt update
sudo apt -y install bzip2
```

#### Zypper

```sh
sudo zypper refresh
sudo zypper install -y lbzip2
```
