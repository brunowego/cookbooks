# Quasar

## Links

- [Code Repository](https://github.com/quasarframework/quasar)
- [Main Website](https://quasar.dev)

## CLI

### Installation

```sh
# Using Yarn
yarn global add @quasar/cli

# Using NPM
npm install @quasar/cli -g
```

### Initialize

```sh
yarn create quasar
# or
npm init quasar
```

### Commands

```sh
quasar -h
```

### Usage

```sh
#
quasar dev

#
quasar build
```
