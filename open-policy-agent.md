## CLI

### Installation

#### Homebrew

```sh
brew install opa
```

### Commands

```sh
opa -h
```

### Usage

```sh
#
opa run \
  -s \
  ./example.rego

#
echo -e '[INFO]\thttp://127.0.0.1:8181'

#
opa test .
opa test --format=json -v .
```
