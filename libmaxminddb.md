### Installation

#### Homebrew

```sh
brew install libmaxminddb
```

#### APT

```sh
sudo add-apt-repository ppa:maxmind/ppa

sudo apt update
sudo apt -y install libmaxminddb-dev

# libmaxminddb0 mmdb-bin
```

#### YUM

```sh
yum check-update
sudo yum -y install libmaxminddb-devel
```
