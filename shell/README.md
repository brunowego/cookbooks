```sh

# Ensure some environment variables are set
: "${BACKEND_URL:?BACKEND_URL not set in environment}"
```

### Server Status Using Shell Script

```sh
#! /bin/bash

SERVER_IP=$1

ping $SERVER_IP -c 2

if [ $? -eq 0]; then
    echo "$SERVER_IP is up"
else
    echo "$SERVER_IP not runnig"
fi
```
