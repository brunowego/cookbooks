# k3s

## Links

- [Main Website](https://k3s.io)

## CLI

### Installation

#### Linux

```sh
curl -sfL 'https://get.k3s.io' | \
  sh -
```

### Commands

```sh
k3s -h
```

### Usage

```sh
#
k3s server

#
k3s agent

#
k3s kubectl

#
k3s crictl

#
k3s ctr

#
k3s check-config

#
k3s etcd-snapshot
```
