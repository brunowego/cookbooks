## CLI

### Installation

#### Homebrew

```sh
brew tap garden-io/garden
brew install garden-cli
```

### Commands

```sh
garden -h
```

### Usage

```sh
#
garden create project

#
garden create module

#
garden deploy

#
garden test

#
garden dev --env=remote

#
garden run workflow full-test
```
