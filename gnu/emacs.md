# GNU Emacs

## CLI

### Installation

#### Homebrew

```sh
brew install emacs
```

### Environment

For Bash or Zsh, put something like this in your `$HOME/.bashrc` or `$HOME/.zshrc`:

```sh
# GNU Emacs
export PATH="$HOME/.emacs.d/bin:$PATH"
```
