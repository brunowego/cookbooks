# NestJS TypeORM

## Links

- [Code Repository](https://github.com/nestjs/typeorm)

## Installation

```sh
# Using Yarn
yarn add @nestjs/typeorm typeorm pg
yarn add ts-node --dev
```

## Bootstrap

```sh
#
npx nest generate module database
npx nest generate module user
```

## Configuration

**Refer:** `./`

// ...
