## CLI

### Execute

```sh
# Using NPX
npx @nestjs/cli -h
```

### Installation

```sh
# Using NPM
npm install -g @nestjs/cli
```

### Commands

```sh
nest -h
```

### Bootstrap

```sh
#
nest new \
  -gs \
  -l TypeScript \
  -p yarn \
  --strict \
  [api-name]
```

### Usage

```sh
#
nest info
```
