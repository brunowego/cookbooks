## CLI

### Commands

```sh
aws waf help
aws wafv2 help
```

### Usage

```sh
#
aws wafv2 list-web-acls \
  --scope 'REGIONAL' \
  --region 'us-east-1'

#
aws wafv2 get-web-acl \
  --scope 'REGIONAL' \
  --region 'us-east-1' \
  --name [web-acl-name] \
  --id [web-acl-id]
```
