## CLI

### Installation

#### Homebrew

```sh
brew install hugo
```

#### APT

```sh
sudo apt update
sudo apt -y install hugo
```

#### Chocolatey

```sh
choco install hugo
```

### Commands

```sh
hugo -h
```

### Bootstrap

```sh
#
hugo new site <name> && cd "$_"

#
hugo new theme <name>
```

### Usage

```sh
#
hugo new <page.md>
hugo new <section>/<page.md>

#
hugo serve
hugo serve -D --cleanDestinationDir

#
hugo --minify
```

### Tips

#### Download Theme

```sh
#
git clone 'https://github.com/EmielH/tale-hugo.git' ./themes/tale

#
cat << EOF | tee -a ./config.toml
theme = 'tale'
EOF

#
hugo --minify
```
