# Devbox

## Links

- [Code Repository](https://github.com/jetpack-io/devbox)
- [Main Website](https://jetpack.io/devbox)

## CLI

### Installation

```sh
curl -fsSL https://get.jetpack.io/devbox | bash
```

### Commands

```sh
devbox help
```

### Usage

```sh
#
devbox init

#
devbox add python@3.10

#
devbox shell
```
