## CLI

### Installation

#### Binary

```sh
# Darwin
wget \
  'https://github.com/pocketbase/pocketbase/releases/download/v0.2.7/pocketbase_0.2.7_darwin_amd64.zip' \
  -O /tmp/pocketbase.zip

( cd /tmp && unzip ./pocketbase.zip -d /usr/local/bin && rm ./pocketbase.zip )
```

### Commands

```sh
pocketbase -h
```

### Usage

```sh
#
pocketbase serve
```
