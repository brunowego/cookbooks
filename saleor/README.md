# Saleor Commerce

**Keywords:** Headless Commerce

## Links

- [Main Website](https://saleor.io)
- [Cloud](https://cloud.saleor.io)

## CLI

### Installation

```sh
npm install @saleor/cli -g
```

### Commands

```sh
saleor -h
```

### Configuration

```sh
#
saleor login
saleor vercel login
saleor github login

#
saleor status
```

### Usage

```sh
#
saleor register

#
saleor organization list

#
saleor project list

#
saleor storefront create --demo
```
