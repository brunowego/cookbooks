## Docker

### Running

```sh
docker run -it --rm \
  $(echo "$DOCKER_RUN_OPTS") \
  -h ksql-cli \
  --name ksql-cli \
  docker.io/confluentinc/cp-ksql-cli:5.3.2 http://127.0.0.1:8088
```

