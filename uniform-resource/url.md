## URL + URN = URI

| Name                        | Abbr. | Example                                |
| --------------------------- | ----- | -------------------------------------- |
| Uniform Resource Identifier | URI   | `https://xyz.tld/user/john.html#posts` |
| Uniform Resource Name       | URN   | `xyz.tld/user/john.html#posts`         |
| Uniform Resource Locator    | URL   | `https://xyz.tld/user/john.html`       |
