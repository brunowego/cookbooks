2. Apple ID
3. iCloud

   - Check: Desktop & Documents Folder
   - Done

1. System Preferences

4. Uncheck True Tone


2. Network
3. Select: Thunderbolt Ethernet
4. Button: Advanced...
5. Tab: 802.1X
6. Uncheck: Enable automatic connection
