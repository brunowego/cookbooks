# go DiskUsage() (gdu)

## CLI

### Dependencies

- Ubuntu
  - [Software Properties Common](/apt/software-properties-common.md#installation)

### Installation

#### Homebrew

```sh
brew install -f gdu

brew link --overwrite gdu
```

### APT

```sh
sudo add-apt-repository ppa:daniel-milde/gdu

sudo apt update
sudo apt -y install gdu
```
