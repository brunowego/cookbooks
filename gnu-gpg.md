# GNU Privacy Guard (GPG) or GNU Pretty Good Privacy (PGP)

## CLI

### Installation

#### Homebrew

```sh
brew install gpg
```

#### APT

```sh
sudo apt update
sudo apt -y install gnupg
```

### Commands

```sh
gpg -h
```

### Issues

#### Not installed

```log
E: gnupg, gnupg2 and gnupg1 do not seem to be installed, but one of them is required for this operation
```

Just install.
