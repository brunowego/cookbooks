# Mockoon

**Keywords:** Mocking, Proxy

## Links

- Docs
  - [CORS](https://mockoon.com/docs/latest/cors/)

## App

### Installation

```sh
# Homebrew
brew install --cask mockoon

# Chocolatey
choco install -y mockoon
```

### Usage

```sh
curl \
  -X GET \
  'http://127.0.0.1:3001/api/users'

curl \
  -X POST \
  'http://127.0.0.1:3001/api/users'
```
