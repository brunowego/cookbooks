## CLI

### Installation

#### NPM

```sh
npm install @angular/cli -g
```

#### Homebrew

```sh
brew install angular-cli
```

### Commands

```sh
ng help
```

### Usage

```sh
#
ng new [app-name]

#
ng serve
```
