## CLI

### Installation

#### Homebrew

```sh
brew install openapi-generator
```

#### Scoop

```sh
scoop install openapi-generator-cli
```

### Commands

```sh
openapi-generator -h
```

### Usage

```sh
#
openapi-generator generate \
  -i <filename>.yml \
  -g go \
  -o ./<foldername>
```
