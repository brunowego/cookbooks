## App

### Installation

#### Homebrew

```sh
brew install --cask 1password
```

#### Chocolatey

```sh
choco install -yf -not-silent 1password
```

## CLI

### Installation

#### Homebrew

```sh
brew install --cask 1password-cli
```

### Commands

```sh
op -h
```

### Usage

```sh
#
op signin [example]
```
