# 1Password

## Links

- [Main Website](https://1password.com/)
- [Dashboard](https://my.1password.com/home)

## App

### Installation

#### Homebrew

```sh
brew install --cask 1password
```

#### Chocolatey

```sh
choco install -yf -not-silent 1password
```

### Issues

#### macOS QR Code Failed

```log
QR code detection failed.
```

Use cell phone instead.

### Installation
