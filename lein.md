## Docker

### Running

```sh
# REPL
docker run -it --rm \
  $(echo "$DOCKER_RUN_OPTS") \
  -h clojure \
  --name clojure \
  docker.io/library/clojure:openjdk-8-lein
```

## CLI

### Installation

#### Homebrew

```sh
brew install leiningen
```

### Commands

```sh
lein help
```

### Usage

```sh
# Version
lein -v

# REPL
lein repl

pkill java
```
