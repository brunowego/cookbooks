# Extend with Google Fonts

## Configuration

**Refer:** `./fonts.css`

```css
@import url('https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&display=swap&subset=latin');

@layer base {
  :root {
    --font-sans: 'Inter', sans-serif;
  }
}
```
