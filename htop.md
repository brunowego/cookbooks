# htop

## CLI

### Installation

#### Homebrew

```sh
brew install htop
```

#### YUM

```sh
yum check-update

# Repo: EPEL
sudo yum -y install htop
```

#### APT

```sh
sudo apt update
sudo apt -y install htop
```

#### APK

```sh
sudo apk update
sudo apk add htop
```

### Commands

```sh
htop -h
```
