## Installation

### Krew

```sh
kubectl krew install view-utilization
```

## Commands

```sh
kubectl view-utilization --help
```

## Usage

```sh
#
kubectl view-utilization -h

#
kubectl view-utilization nodes -h

#
kubectl view-utilization namespaces -h
```
