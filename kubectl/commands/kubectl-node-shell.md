# kubectl-node-shell

## Links

- [Code Repository](https://github.com/kvaps/kubectl-node-shell)

## Installation

### Krew

```sh
kubectl krew index add kvaps 'https://github.com/kvaps/krew-index'
kubectl krew install kvaps/node-shell
```

## Usage

```sh
#
kubectl get nodes

#
kubectl node_shell [node-name]
```
