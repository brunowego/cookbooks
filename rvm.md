# RVM

## CLI

### Dependencies

#### APT

```sh
sudo apt update
sudo apt -y install build-essential libreadline-dev libssl-dev curl libsqlite3-dev
```

### Installation

#### APT

```sh
curl -L get.rvm.io | /bin/bash -s stable --rails
```

### Usage

#### Init

```sh
source $HOME /.rvm/scripts/rvm
```
