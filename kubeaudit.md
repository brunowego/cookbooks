# kubeaudit

## Links

- [Code Repository](https://github.com/Shopify/kubeaudit)

## CLI

### Installation

#### Homebrew

```sh
brew install kubeaudit
```

### Commands

```sh
kubeaudit -h
```

### Usage

```sh
#
kubeaudit all \
  -f '/path/to/manifest.yml'

#
kubeaudit autofix \
  -f '/path/to/manifest.yml'
```
