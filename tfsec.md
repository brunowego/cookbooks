# tfsec

**Keywords:** Terraform Security Scanner

## Links

- [Code Repository](https://github.com/tfsec/tfsec)

## CLI

### Installation

#### Homebrew

```sh
brew install tfsec
```

#### Chocolatey

```sh
choco install -y tfsec
```

### Commands

```sh
tfsec -h
```

### Usage

```sh
#
tfsec ./
```

### Tips

#### Visual Studio Code

```sh
code --install-extension tfsec.tfsec
```
