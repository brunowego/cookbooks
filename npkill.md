# NPKILL

**Keywords:** Remove `./node_modules` recursively, Node module remover

## Links

- [Code Repository](https://github.com/voidcosmos/npkill)
- [Main Website](https://npkill.js.org/)

## CLI

### Installation

```sh
# Using NPM
npm install npkill -g

# Using Yarn v1.x
yarn global add npkill

# Using Bun
bun add npkill -g
```

### Commands

```sh
# Using NPX
npx npkill -h

# Using pnpm
pnpm dlx npkill -h

#
bunx npkill -h

# Using local
npkill -h
```

### Usage

```sh
#
npkill -d ./
```
