## CLI

### Installation

#### Homebrew

```sh
brew tap tailwarden/komiser
brew install komiser
```

#### Darvin Binary

```sh
curl \
  -L https://github.com/tailwarden/komiser/releases/download/v3.0.18/komiser_Darwin_x86_64 \
  -o /usr/local/bin/komiser && \
    sudo chmod +x /usr/local/bin/komiser
```

### Commands

```sh
komiser -h
```
