## CLI

### Installation

```sh
# Using NPM
npm install @medusajs/medusa-cli -g
```

### Commands

```sh
#
medusa -h

# Using NPX
npx @medusajs/medusa-cli@latest -h
```

### Usage

<!--
medusa user -e brunowego@gmail.com --invite
-->

```sh
#
medusa telemetry --disable

#
medusa migrations run

#
medusa seed -f ./data/seed.json

#
medusa develop

#
medusa start
```
