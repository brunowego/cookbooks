# Semantic HTML

## Syntax

```html
<!DOCTYPE html>
<html lang="en">
  <head />
  <body>
    <header>
      <nav>
        <a href="...">...</a>
      </nav>
    </header>

    <aside />

    <main>
      <article>
        <header>
          <h1>...</h1>

          <p>...</p>
        </header>

        <section>
          <h2>...</h2>

          <p>...</p>
        </section>

        <footer>
          <p>...</p>
        </footer>
      </article>
    </main>

    <footer>
      <p>...</p>
    </footer>
  </body>
</html>
```
