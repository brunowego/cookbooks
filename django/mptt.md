## Library

### Installation

#### pip

```sh
# As user dependency
pip3 install -U django-mptt

# As project dependency
echo 'django-mptt==0.11.0' >> ./requirements.txt
```

Add `django_extensions` to your `INSTALLED_APPS` setting.

```py
INSTALLED_APPS = [
    # ...
    'django_mptt_admin',
]
```
