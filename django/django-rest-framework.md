## Library

### Installation

#### pip

```sh
# As user dependency
pip3 install -U djangorestframework

# As project dependency
echo 'djangorestframework==3.12.2' >> ./requirements.txt
```

Add `rest_framework` to your `INSTALLED_APPS` setting.

```py
INSTALLED_APPS = [
    # ...
    'rest_framework',
]
```
