## CLI

### References

- [Getting Started](https://guides.cocoapods.org/using/getting-started.html)

### Installation

#### Homebrew

```sh
brew install cocoapods
```

#### RubyGems

```sh
gem install cocoapods
```

### Commands

```sh
pod -v
```

### Usage

```sh
# Version
pod --version

#
pod init

#
pod setup

#
pod repo update master

# Install Dependencies
pod install
```
